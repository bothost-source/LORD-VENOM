## Installation

### Termux (Android)

```bash
# Update packages
pkg update && pkg upgrade -y

# Install required packages
pkg install nodejs python git -y

# Install Python dependencies
pip install flask

# Clone repository
git clone https://github.com/bothost-source/LORD-VENOM.git

# Enter directory
cd LORD-VENOM

# Install Node.js dependencies
npm install

# Start the bot
bash start.sh
